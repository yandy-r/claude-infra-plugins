etc.) pick up the same rules without duplicating prose.

- AI-tool-neutral mirror at `.ai/rules/project.md` for runtimes that follow the convention.

- For deep analysis of this repository, start with `CLAUDE.md` and the stack overview.
- yci is Claude-native only in Phase 0 — do not attempt to generate Cursor / Codex / opencode
  bundles. See `CLAUDE.md` § Scope.
- yci-specific policy (non-goals, compliance-adapter pattern, customer-data rule) lives in
  [`yci/CONTRIBUTING.md`](yci/CONTRIBUTING.md).
